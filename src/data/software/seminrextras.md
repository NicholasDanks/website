seminrExtras is a companion package to SEMinR that provides additional utilities, experimental features, and extended functionality for researchers working with structural equation models in R.

## Features

- **Threshold-Based Model Evaluation**: Automated evaluation of model quality criteria against established thresholds from the PLS-SEM literature, streamlining the reporting process.
- **Advanced Reporting**: Generate comprehensive model summary reports, including formatted tables suitable for academic publication.
- **Experimental Methods**: Early access to cutting-edge SEM techniques and evaluation criteria that are under active development before potential inclusion in the core SEMinR package.
- **Helper Utilities**: Convenience functions for common tasks such as data preparation, indicator reordering, and results extraction that complement the core SEMinR workflow.

seminrExtras serves as both a proving ground for new ideas and a collection of practical tools that extend the core SEMinR ecosystem, enabling researchers to go further with their SEM analyses.
